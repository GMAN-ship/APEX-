Ghost-Master Terminal

Complete Production Blueprint: AI Prompts & Native Android APK Export Guide

To turn your Lovable / Supabase / MT5 trading dashboard into a downloadable Android .apk, use a two-tier approach: prepare the web assets for native wrapping, then compile via a cloud or local builder (Capacitor / Cordova / Flutter Webview).

Architecture Concept: Treat the Lovable project as a production PWA/SPA and bundle the compiled static assets into a native wrapper to gain native capabilities like push, wake locks, and native distribution.

Phase 1: Production Build Master Prompt (run inside Lovable)

Before compiling, run accessibility and mobile-optimization passes: touch target sizing, viewport rules, offline fallbacks, and asset packaging readiness.

Act as a Lead Mobile PWA Architect. Freeze code for an Android APK export and ensure: 1. TOUCH & VIEWPORT OPTIMIZATION: Ensure all actionable controls are >=48px target. Replace scrollbars with native momentum scrolling and disable double-tap zoom. 2. ASSET PACKAGING: Produce a single static output folder (/dist or /build) with relative paths and bundled SVG assets. 3. OFFLINE STANDBY: Provide cached metrics and block order execution while offline; show a clear reconnect modal.

Phase 2: Android APK Compilation Blueprint

Choose a compilation method:

Cloud BuildGit + VoltBuilder / CodeMagicNo local Android Studio3–5 min

Method A: Cloud Compilation (prompt for cloud builder)

Generate a production config.xml / voltbuilder.json with: - package: com.ghostmaster.terminal.app - name: Ghost-Master Terminal - version: 1.0.0 - orientation: portrait - permissions: network, wake-lock, notifications Bundle /dist and zip for the cloud compiler.

Deployment Note: Upload the /dist zip to VoltBuilder/CodeMagic to receive an APK.

Method B: Local CLI (Capacitor)

Bash script outline: 1. npm install @capacitor/core @capacitor/cli 2. configure capacitor.config.json (webDir -> /dist) 3. npx cap add android 4. npx cap open android (then Build -> Build APK(s))

Phase 3: Installation Verification

  • Enable Install Unknown Apps on the Android device.
  • Bypass Play Protect if necessary (Install Anyway).
  • Verify Wake Lock by running the engine for 15 minutes and confirming live metric sync.
